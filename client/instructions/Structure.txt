Back-end Workflow

1. Users are required to use search bar to find the appropriate sample tests: By typing the name (AutoComplete), it will show up the dropdown selection

Example:
+ Calculus
+ Linear Algebra
+ Probability
+ And so on

2. Once selecting the following sections, users need to select respective sections: University, Year, Semester and Type. There are some cases:
+ There's only one-test â†’ Automatically filters out
+ There's only one university, but multiple choices in remaining sections
+ There are multiple universities, and multiple choices in remaining sections

3. After filtering out the appropriate sections, a Google Drive embedded link (viewer) is shown up below the search and section bars.

- Extras:
+ Google Drive embed (iframe)
+ localStorage (client-side)
+ REST API JSON
